# 📊 Painel de Marketing — Instagram & Facebook

Pipeline automatizada com Python + Meta Graph API integrada ao Power BI.

## 🎯 Sobre o Projeto

Solução desenvolvida para monitoramento de performance de redes sociais,
eliminando a extração manual de dados e garantindo atualização diária
automática no Power BI.

O dashboard foi projetado para atender dois perfis de usuário:
o time de marketing e gestores sem familiaridade técnica — por isso,
cada indicador conta com orientações de interpretação integradas ao painel.

## ⚙️ Como funciona

1. Script Python autentica na Meta Graph API
2. Extrai métricas de Instagram e Facebook automaticamente
3. Dados são carregados no Power BI via arquivo estruturado
4. Dashboard atualiza diariamente sem intervenção manual

## 📈 Métricas monitoradas

- Número de seguidores
- Alcance e impressões
- Taxa de engajamento
- Interações por publicação
- Visualizações de perfil
- Cliques na bio
- Performance por tipo de publicação (imagem, carrossel, vídeo)

## 🛠️ Stack

- Python
- Meta Graph API (Facebook & Instagram)
- Power BI
- Power Query
- DAX
